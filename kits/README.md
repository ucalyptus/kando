# Kits

A kit is a domain bundle: object types, responders, tools, prompts, and
policies packaged for a specific use case.

Kando has no opinion about what a "task" or "belief" or "claim" is — that
is domain vocabulary defined by the kit. The runtime only knows events.

| Kit | Status | Description |
|---|---|---|
| `diligence` | Phase 1 | Structured due-diligence on companies/entities |
| `research` | Planned | Research decomposition and synthesis |
